# Tienda Virtual de PCs

Proyecto web desarrollado con HTML, CSS y JavaScript que simula una tienda de computadores.

## 🚀 Funcionalidades
- Catálogo de productos
- Carrito de compras
- Interfaz interactiva con JavaScript
- Diseño responsive

## 🛠️ Tecnologías
- HTML
- CSS
- JavaScript
